## Installation Guide

This guide will walk you through installing EternalCombat on your Minecraft server.

### Requirements

| Requirement | Details |
| :--- | :--- |
| **Minecraft Version** | 1.17.1 - 1.21.1 |
| **Java Version** | Java 17 or newer |
| **Server Software** | Paper, Spigot, or compatible forks |
| **Required Plugin** | PacketEvents |

<Callout type="danger" title="PacketEvents Required">
  EternalCombat **will not work** without PacketEvents installed. Make sure to install it first!
</Callout>

### Installation steps

<Steps>
  <Step title="Install PacketEvents" icon="Package">
    Download PacketEvents from one of these sources:

    - [Modrinth](https://modrinth.com/plugin/packetevents) (Recommended)
    - [SpigotMC](https://www.spigotmc.org/resources/packetevents-api.80279/)
    - [GitHub](https://github.com/retrooper/packetevents/releases)

    Place the JAR file in your server's `plugins` folder.
  </Step>

  <Step title="Download EternalCombat" icon="Download">
    Download the latest version of EternalCombat from:

    - [Modrinth](https://modrinth.com/plugin/eternalcombat) (Recommended)
    - [SpigotMC](https://www.spigotmc.org/resources/eternalcombat.109056/)
    - [Hangar](https://hangar.papermc.io/EternalCodeTeam/eternalcombat)
    - [GitHub Releases](https://github.com/EternalCodeTeam/EternalCombat/releases)

    Place the JAR file in your server's `plugins` folder.
  </Step>

  <Step title="Start Your Server" icon="Play">
    Start or restart your Minecraft server. You should see EternalCombat loading in the console:

    ```
    [EternalCombat] Loading EternalCombat v2.4.0
    [EternalCombat] Enabling EternalCombat v2.4.0
    ```
  </Step>

  <Step title="Configure the Plugin" icon="Settings">
    After the first start, EternalCombat will generate its configuration file. Edit it to customize the plugin:

    ```
    plugins/EternalCombat/config.yml
    ```

    See the [Configuration](/docs/eternalcombat/configuration) page for detailed information about all options.
  </Step>
</Steps>

### Generated files

After starting the server, EternalCombat will create the following structure:

<FileTree>
  <FileTreeItem name="plugins" type="directory" defaultOpen>
    <FileTreeItem name="EternalCombat" type="directory" defaultOpen>
      <FileTreeItem name="config.yml" type="file" />
    </FileTreeItem>
  </FileTreeItem>
</FileTree>

### Updating the plugin

<Steps>
  <Step title="Backup configuration" icon="Save">
    Before updating, always backup your configuration file:

    <Command variant="bash" theme="light">
      cp plugins/EternalCombat/config.yml plugins/EternalCombat/config.yml.backup
    </Command>
  </Step>

  <Step title="Download a new version" icon="Download">
    Download the latest version from [Modrinth](https://modrinth.com/plugin/eternalcombat) or other sources.
  </Step>

  <Step title="Replace JAR file" icon="RefreshCw">
    1. Stop your server
    2. Delete the old `EternalCombat-*.jar` file
    3. Place the new JAR file in the `plugins` folder
    4. Start your server
  </Step>
</Steps>

<Callout type="tip" title="Configuration Migration">
  EternalCombat automatically migrates your configuration when updating. Your settings will be preserved, and new options will be added with their default values.
</Callout>
