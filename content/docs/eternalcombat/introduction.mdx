## Welcome to EternalCombat

EternalCombat is a powerful combat logging plugin that prevents players from escaping fights by logging out. It provides a complete PvP management system with visual indicators, spawn protection, and extensive customization options.

<Callout type="warning" title="Required Dependency">
  EternalCombat requires **PacketEvents** to work properly. Make sure to install it before using EternalCombat.
  
  [Download PacketEvents from Modrinth](https://modrinth.com/plugin/packetevents)
</Callout>

### Key features

| Feature | Description |
| :--- | :--- |
| **Combat Logging Prevention** | Players cannot escape fights by disconnecting - they will be punished |
| **Visual Combat Border** | Beautiful animated border shows protected regions during combat |
| **Crystal PvP Support** | Full support for end crystals and respawn anchors |
| **Spawn Protection** | Prevent players from fleeing to safe zones during combat |
| **Customizable Restrictions** | Block commands, elytra, flying, inventory, and more during combat |
| **PlaceholderAPI Support** | Display combat status in scoreboards and other plugins |
| **WorldGuard Integration** | Automatic protection for regions with PvP disabled |
| **Developer API** | Easy-to-use API for custom integrations |

### Compatibility

<Badge variant="success">Minecraft 1.17.1 - 1.21.1</Badge> <Badge variant="info">Java 17+</Badge> <Badge variant="secondary">Paper/Spigot</Badge>

### Quick start

<Steps>
  <Step title="Install PacketEvents" icon="Download">
    Download PacketEvents from [Modrinth](https://modrinth.com/plugin/packetevents) and place it in your `plugins` folder.
  </Step>
  <Step title="Install EternalCombat" icon="Package">
    Download EternalCombat from [Modrinth](https://modrinth.com/plugin/eternalcombat), [SpigotMC](https://www.spigotmc.org/resources/eternalcombat.109056/), or [Hangar](https://hangar.papermc.io/EternalCodeTeam/eternalcombat) and place it in your `plugins` folder.
  </Step>
  <Step title="Start Your Server" icon="Play">
    Start or restart your server. EternalCombat will generate its configuration file automatically.
  </Step>
  <Step title="Configure" icon="Settings">
    Edit `plugins/EternalCombat/config.yml` to customize the plugin to your needs.
  </Step>
</Steps>
