<Callout type="warning" title="Required Dependency">
  EternalCombat requires **PacketEvents** to work properly. Make sure to install it before using EternalCombat.

  [Download PacketEvents from Modrinth](https://modrinth.com/plugin/packetevents)
</Callout>

### Key features

| Feature | Description |
| :--- | :--- |
| **Combat Logging Prevention** | Players cannot escape fights by disconnecting - they will be punished |
| **Visual Combat Border** | Beautiful animated border shows protected regions during combat |
| **Crystal PvP Support** | Full support for end crystals and respawn anchors |
| **Region Protection** | Prevent players from fleeing to safe zones during combat |
| **Customizable Restrictions** | Block commands, elytra, flying, inventory, and more during combat |
| **PlaceholderAPI Support** | Display combat status in scoreboards and other plugins |
| **WorldGuard Integration** | Automatic protection for regions with PvP disabled |
| **Developer API** | Easy-to-use API for custom integrations |
