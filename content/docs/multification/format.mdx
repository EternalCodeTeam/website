## Quick rules

- ‚ú® **Quote strings** that contain `<colors>` or spaces.
- üìã **Multi‚Äëline chat** = YAML list (`- "line"`).
- ‚è±Ô∏è **Times**: `times: "fadeIn stay fadeOut"` uses seconds, e.g. `"0.5s 3s 1s"`.

## Chat

<CodeTabs>
  <CodeTab label="Single line">

```yaml
welcome: "<green>Hello, {player}!"
```

  </CodeTab>
  <CodeTab label="Multiple lines">

```yaml
welcome:
  - "<gold>Hey {player}!"
  - "<yellow>Enjoy your stay"
```

  </CodeTab>
</CodeTabs>

## Actionbar

```yaml
healSuccess:
  actionbar: "<green>‚ù§ Health restored"
```

## Title & Subtitle

```yaml
warning:
  title: "<red>Warning!"
  subtitle: "<gray>PvP enabled in this area"
  times: "0.5s 3s 1s" # fade in, stay, fade out
```

## Bossbar

```yaml
bossInfo:
  bossBar:
    color: "GREEN"
    text: "<green>Objective updated"
    progress: 0.6
```

## Sounds

<CodeTabs>
  <CodeTab label="Basic">

```yaml

  sound: "ENTITY_PLAYER_LEVELUP 2.0 1.0" # volume, pitch
```

  </CodeTab>
  <CodeTab label="With category">

```yaml

  # Sound categories: https://hub.spigotmc.org/javadocs/spigot/org/bukkit/SoundCategory.html
  sound: "PLAYER_LEVELUP WEATHER 2.0 1.0"
```

  </CodeTab>
</CodeTabs>
