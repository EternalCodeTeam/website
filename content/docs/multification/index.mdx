**Multification** is a library for sending notifications in Minecraft plugins. Built on Adventure API, it provides type-safe, configurable messaging.

<Callout type="tip" title="Why Multification?">
  Clean, type-safe API for sending chat, titles, actionbars, bossbars, and sounds â€“ all from a single, configurable
  `Notice` object.
</Callout>

## Quick Start

```java
// 1. Extend platform class
public class NoticeService extends PaperMultification<MessagesConfig> {
    private final MessagesConfig config;

    @Override
    protected TranslationProvider<MessagesConfig> translationProvider() {
        return locale -> config;
    }

    @Override
    protected ComponentSerializer<Component, Component, String> serializer() {
        return MiniMessage.miniMessage();
    }

    @Override
    protected AudienceConverter<CommandSender> audienceConverter() {
        return new PaperAudienceConverter();
    }
}

// 2. Send messages
noticeService.create()
    .player(playerUUID)
    .notice(config -> config.welcomeMessage)
    .placeholder("{player}", player.getName())
    .send();
```
