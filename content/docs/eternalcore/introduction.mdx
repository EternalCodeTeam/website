## How does it work?

EternalCore is a plugin for Minecraft servers that includes most basic functions and commands that everyone needs. It's easy to use, configure, and very lightweight.

Key features:
- Teleportation (TPA, RTP, Warps, Homes)
- Chat management
- Essential commands (God, Fly, Heal, Feed)
- And much more!
