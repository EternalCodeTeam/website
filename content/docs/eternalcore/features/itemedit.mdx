## ‚úèÔ∏è Item Edit ‚Äî modify item properties
The `/itemedit` command allows you to modify various properties of the item you are holding, such as its name, lore, and flags.
This is useful for customizing items for gameplay or aesthetic purposes.

### ‚úÖ Usage

```text
/itemedit name <new_name>
/itemedit lore <line_number> <new_lore>
/itemedit flag <item_flag>
```

<AlertBox type="info">
  When running `/itemedit flag` the specified flag will be toggled off for the held item.
  Re-run the command to toggle it back on.
</AlertBox>
