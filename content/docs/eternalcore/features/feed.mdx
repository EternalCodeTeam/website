## Introduction

The `/feed` command fills the hunger bar of the executor or a specified player. Useful for quickly restoring food without giving items.

## Usage

```text
/feed [player]
```
- Omitting `[player]` feeds the command sender.
- Using `[player]` feeds the target player.

## Examples

```text
/feed
→ Your hunger has been restored to full

/feed Alex
→ Alex's hunger has been restored
```
