## 🍗 Feed — refill hunger

The `/feed` command fills the hunger bar of the executor or a specified player. Useful for quickly restoring food without giving items.

### ✅ Usage

```text
/feed [player]
```

- Omitting `[player]` feeds the command sender.
- Using `[player]` requires `eternalcore.feed.other` permission.

### 🎯 Examples

```text
/feed
→ Your hunger has been restored to full

/feed Alex
→ Alex's hunger has been restored (staff only)
```

## ℹ️ Notes

- Permission: `eternalcore.feed` (self), `eternalcore.feed.other` (other players).
- This command only affects the hunger/saturation values, not health.
