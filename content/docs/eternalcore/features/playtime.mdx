The `/playtime` command allows you to track and display the total playtime of players on your Minecraft server.
This is useful for server admins and players who want to monitor their time spent in the game.
Required permissions:
- `eternalcore.playtime` (self)
- `eternalcore.playtime.other` (others)

### ‚úÖ Usage

```
/playtime [player]
```
