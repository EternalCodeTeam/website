## Introduction

The `/clear` command removes items from a player's inventory. It can be used to clear everything, remove specific item types, or target a specific player.

## Usage

```text
/clear
→ Clears your entire inventory

/clear <player>
→ Clears the specified player's inventory
```

<AlertBox type="warning" title="Be careful!">
  Inventory clearing is a potentially destructive action and cannot be undone in-game.
</AlertBox>
