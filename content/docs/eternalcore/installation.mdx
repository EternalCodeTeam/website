## Installation

### Step 1: Download the plugin
After downloading the EternalCore JAR file, place it in the `plugins` folder on your Minecraft server. Make sure the file is located in the same directory as other plugins.

### Step 2: Restart the server
Restart the Minecraft server. The EternalCore plugin should be automatically loaded.

<AlertBox type="info" title="Support">
  If the plugin doesn't load correctly, inform us about it in the [Issues](https://github.com/EternalCodeTeam/EternalCore/issues) tab or on our [Discord](https://discord.gg/FQ7jmGBd6c).
</AlertBox>

## Upgrading

<AlertBox type="warning" title="Backup">
  Before starting the upgrade, make sure you back up your EternalCore plugin configuration to avoid potential configuration conflicts.
</AlertBox>

### Step 1: Download the latest version
Download the latest version of EternalCore from the project website.

### Step 2: Replace the JAR file
Delete the old JAR file and place the new one in the `plugins` folder.

### Step 3: Restart the server
Restart your Minecraft server to load the new version.

### Step 4: Verify the update
Check if the plugin works properly and all functionality is available.

## Warning

<AlertBox type="danger" title="Reloading">
  We recommend **not** using the `/reload` command or plugins reloading in real-time (e.g. PlugMan), as they may cause problems with plugins and lead to unexpected errors.
</AlertBox>
